Python Projects

As for Python, I've been studying it for about 8 months at this point.


My Python knowledge consists of: variable types, if statemens, for loops, while loops, lists, dictionaries (and comprehension of course), and most importantly, OOP.

That's not all, though.

I can use libraries such as: Tkinter, Requests, Pandas, Selenium, BeautifulSoup, Datetime, Random and SMTP.

Last but not least, I am currently studying back-end development, using the Flask framework and Jinja.


As for the projects, I created several web scraping ones that collect data and then enter said data in *.db files using SQLite.

But the most complex project that I created is a weather app that I built from scratch.

It collects data from 2 APIs (MeteoRomania and OpenWeatherMap) using the Requests module and then renders the results inside a GUI that I created with Tkinter.

It basically allows you to select a location (in Romania) and then displays the current temperature and cloud coverage. And the project is structured using OOP.
